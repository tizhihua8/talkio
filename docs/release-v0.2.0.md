## What's New

### ✨ 单聊切换模型
聊天过程中可以随时切换模型，无需新建对话。输入框左下角显示当前模型名称，点击即可切换，对话标题自动同步更新。

### 🤖 托管讨论（Auto-pilot）重新设计
群聊中的托管讨论交互全面重做：
- 输入框上方展开轮次选择器，支持 3 / 5 / 10 / 20 轮
- 托管进行中，输入区域替换为控制面板，显示进度条和当前轮次
- 一键停止，交互更清晰直观

### 🎨 输入栏 UI 优化
- 操作图标增大（22px 图标 + 40px 触控区域），更容易点击
- 所有按钮统一为圆形，视觉一致
- 移除不常用的快捷提示功能，界面更简洁

### 🔧 模型选择器重构
- 按供应商分组显示，与模型列表页风格一致
- 支持 Android 返回键关闭
- 修复底部模型被导航栏遮挡的问题

### 🌙 暗色模式适配
输入栏所有硬编码颜色替换为主题色，暗色模式下显示正确。
